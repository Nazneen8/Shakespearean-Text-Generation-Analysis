# Shakespearean-Text-Generation-Analysis

### Overview
This project implements a chaacter-level Recurrent Neural Network (RNN) using PyTorch to generate text in the style of William Shakespeare. The model is trained on the Tiny Shakespeare dataser and learns to predict the next charcter in a sequence. 

### Objectives
- Build a simple RNN using PyTorch
- Train the model on Shakespeare text data
- Implement *one-hot encoding* for input sequences and targets
- Generate 100 words
